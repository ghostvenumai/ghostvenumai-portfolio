# GhostVenumAI (Demo Version)

This folder contains a **simplified demo** of my project **GhostVenumAI**.

The real GhostVenumAI:

- runs network scans,
- parses the results,
- uses AI (GPT-style models) for reasoning,
- and generates human-readable security insights.

This public demo is intentionally reduced to:

- show my **architecture and thinking**,
- demonstrate my **coding style**,
- and illustrate how AI can be integrated into security workflows.

## Files

- `main_example.py`  
  Minimal entry point that simulates a scan and calls the analysis module.

- `analysis_module_example.py`  
  Shows how scan data could be processed and classified (rule-based in this demo).

- `requirements.txt`  
  Example dependencies (simplified) used in the real project.

## Note

This is **not** the full GhostVenumAI product.  
The demo is meant for **portfolio and recruiting** purposes only.
