# GhostPoster (AI Marketing Automation - Demo)

GhostPoster is a concept for automating social media workflows with AI.

Real features planned in the private full version:

- Prepare platform-specific posts based on a topic
- Generate text using AI
- Schedule posts
- Send them through APIs (Twitter/X, Reddit, GitHub)

This public demo:

- does not call any real APIs,
- only builds "post objects",
- and simulates AI-generated content.

Status: **in active development**.
