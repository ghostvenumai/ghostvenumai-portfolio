# GhostShield (AI Malware / Risk Classification - Demo)

GhostShield is an experimental idea to use AI to help:

- classify suspicious files,
- estimate rough risk levels,
- and explain risks in human-readable text.

This public demo does **not**:
- execute real files,
- scan real systems,
- or include any dangerous behaviour.

It only works with small simulated "file metadata" dictionaries to show
how the logic and explanations could look.

Status: **in active development** in the private version.
