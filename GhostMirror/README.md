# GhostMirror (AI Network Traffic Analysis - Demo)

GhostMirror is a concept for AI-assisted network visibility.

Core idea:

- observe network events,
- highlight unusual patterns,
- send selected suspicious events to an AI model (e.g. GPT-5.1),
- and receive human-readable explanations of what might be happening.

This demo:

- does not capture real packets,
- uses simulated events,
- and focuses on how I would structure the logic.

Status: **in active development** in the private version.
